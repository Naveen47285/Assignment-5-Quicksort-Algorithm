# Quicksort Assignment

## Overview
This project implements deterministic and randomized versions of the Quicksort algorithm and compares their performance.

## Files
- quicksort.py → Deterministic version
- randomized_quicksort.py → Randomized version
- performance_test.py → Performance comparison

## How to Run

Run deterministic:
python quicksort.py

Run randomized:
python randomized_quicksort.py

Run performance analysis:
python performance_test.py

## Summary
- Deterministic Quicksort performs well on random data but degrades on sorted inputs.
- Randomized Quicksort maintains consistent performance across all input types.

## Results
Performance graphs are included showing comparison between deterministic and randomized Quicksort under different input conditions.
